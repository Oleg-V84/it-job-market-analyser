# IT Job Market Analyzer

## Description

Analysis of IT job vacancies using Pandas and Matplotlib.
The project explores job demand, locations, remote work opportunities, and employment types.

## Software and libraries being used

- Python 3.14
- Pandas 3.03
- Matplotlib 3.10.9
- Jupyter Notebook 4.5.7

## Project structure

data/ - source data
notebooks/ - analysis
reports/ - findings
src/ - utility code

## Conclusion

1. Data Scientist and UI/UX Designer are the most in-demand roles (both 17.4%), following Software Developer 
17.3%.

2. The majority of vacancies are in London,UK (79 vacancies).

3. Remote vacancies account for 39.7%.

4. Full time positions dominate the market (846 vacancies).

## Source of data

https://www.kaggle.com/datasets/moyukhbiswas/job-postings-dataset
